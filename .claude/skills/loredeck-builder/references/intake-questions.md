# Intake Question Framework

Use the user's `/grill` skill when available (hand it this list as the brief). Otherwise ask these yourself, a few at a time, with concrete options — not as one wall of questions. Stop when you can write the scope brief without guessing.

## Must answer before the brief

1. **Canon and sources.** Which fandom, and which exact sources count? (e.g. "the novel only", "seasons 1–3", "the games, not the anime"). What is explicitly excluded?
2. **Continuity and adaptation.** Book vs film vs game continuity; which canon tier (`primary`, `secondary`, `legends`, `expanded`, `fanon`, `au`)? If sources conflict, which wins?
3. **Story span.** Where does coverage start and stop? Is post-canon / epilogue material in scope?
4. **Deck split preference.** One deck or a family? If a family: split by era, by faction, or by source? (Recommend using the sizing rubric; the user decides.)
5. **Spoiler philosophy.** Which reveals/secrets must stay gated, and how strictly? Is the user's story likely to start mid-canon (needs tight windows) or post-canon (gating matters less)?
6. **Granularity.** Compact / focused / dense / scene-dense (define them briefly). Any hard card-count limit?

## Useful if time allows

7. **Story use.** What kind of scenes will this deck serve (romance, war campaign, slice-of-life, canon-divergent AU)? This shifts which categories deserve density.
8. **POV characters.** Whose knowledge boundaries matter most? (Drives `knowledge`/`secret` cards.)
9. **Source material access.** Can the user supply notes/summaries (user_supplied evidence), or should research be web-based? Any preferred wikis?
10. **AU/crossover intent.** Pure canon, or deliberate deviations to encode as `au` cards?

## Output

Summarize the answers as the intent summary (canon, sources, continuity, span, split, spoiler policy, granularity, sourcing) and present it for confirmation — that confirmation is the intake gate.
